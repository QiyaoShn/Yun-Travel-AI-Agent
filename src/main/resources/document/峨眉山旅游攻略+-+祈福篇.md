# 峨眉山祈福指南

## 峨眉山佛教文化

峨眉山是中国佛教四大名山之一，素有"佛国天堂"之称，是普贤菩萨的道场。

### 历史沿革
- 公元1世纪：佛教传入峨眉山
- 东晋时期：开始建寺
- 唐宋时期：达到鼎盛
- 明清时期：大规模重建

### 主要寺庙
1. 报国寺：峨眉山门户
2. 伏虎寺：森林古刹
3. 清音阁：山水寺庙
4. 洪椿坪：千年古寺
5. 仙峰寺：财神庙
6. 洗象池：佛教圣地
7. 乐山大佛：世界最大石刻弥勒佛

## 祈福攻略

### 金顶祈福
- **最佳时间**：初一、十五
- **祈福内容**：学业、事业、平安
- **必做事项**：
  - 参拜四面十方普贤金像
  - 烧香祈福
  - 撞钟祈愿

### 万年寺祈福
- **最佳时间**：任意时间
- **祈福内容**：求子、延寿
- **必做事项**：
  - 参拜无梁砖殿
  - 祈福求子

### 报国寺祈福
- **最佳时间**：全年
- **祈福内容**：平安、健康
- **必做事项**：
  - 礼佛参拜
  - 请香祈福

## 祈福礼仪

### 进香礼仪
1. 进入寺庙前先净手
2. 选择合适规格的香
3. 点香时用左手（因为右手杀戮）
4. 举香齐眉，心中默念愿望
5. 先拜四方，再拜主殿

### 参拜礼仪
1. 衣冠整洁，端庄得体
2. 不大声喧哗
3. 不踩门槛
4. 顺时针绕佛塔
5. 心诚则灵

### 注意事项
- 不要对着佛像拍照
- 不要用手指点佛像
- 保持安静，尊重他人
- 不带荤腥食物入寺

## 特色祈福活动

### 春节祈福
- 时间：农历正月初一至十五
- 特色：烧头香、撞钟、祈福法会
- 人流量：最大

### 观音诞辰
- 时间：农历二月十九
- 特色：观音法会、放生
- 地点：报国寺

### 普贤菩萨圣诞
- 时间：农历二月廿一
- 特色：金顶法会、点灯祈福
- 地点：金顶

### 地藏王菩萨圣诞
- 时间：农历七月三十
- 特色：地藏法会、祭祖
- 地点：各寺庙
