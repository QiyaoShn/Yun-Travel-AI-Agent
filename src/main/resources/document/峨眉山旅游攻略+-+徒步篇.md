# 峨眉山徒步攻略

## 徒步路线推荐

### 经典两日徒步路线

#### 第一天：报国寺 - 清音阁 - 洪椿坪 - 仙峰寺
- **路程**：约15公里
- **难度**：中等
- **预计时间**：6-8小时
- **特色**：体验"峨眉山的绿"

**沿途看点**：
- 报国寺：峨眉山门户
- 伏虎寺：森林古刹
- 清音阁：山水精华
- 一线天：峡谷奇观
- 生态猴区：野生灵猴

#### 第二天：仙峰寺 - 洗象池 - 金顶
- **路程**：约20公里
- **难度**：较高
- **预计时间**：8-10小时
- **特色**：登顶观日出

**沿途看点**：
- 仙峰寺：财神庙
- 洗象池：传说中普贤菩萨洗象之处
- 雷洞坪：最后的休息站
- 太子坪：观赏金顶的最后位置
- 金顶：峨眉山最高峰

---

## 徒步装备建议

### 必备物品
- 登山杖：减轻膝关节压力
- 舒适的登山鞋：防滑耐磨
- 足够的饮用水：2-3瓶
- 高热量零食：巧克力、能量棒等
- 雨具：山区天气多变
- 保暖衣物：山顶温度低

### 注意事项
1. 提前了解天气情况
2. 不要单独行动
3. 保护环境，不乱扔垃圾
4. 不要喂食野生猴子
5. 注意人身和财物安全

## 徒步时间规划

### 春季（3-5月）
- 气温适宜，10-20℃
- 杜鹃花盛开
- 建议穿着：轻便春装+外套

### 夏季（6-8月）
- 雨水较多，15-28℃
- 可避暑
- 建议穿着：速干衣物+雨具

### 秋季（9-11月）
- 秋高气爽，10-22℃
- 红叶最美
- 建议穿着：秋装+保暖外套

### 冬季（12-2月）
- 可能下雪，0-10℃
- 可观赏雪景
- 建议穿着：羽绒服+防滑鞋

## 住宿建议

### 山下住宿
- 报国寺周边：经济实惠，选择多样
- 峨眉山市区：设施完善

### 山中住宿
- 洪椿坪：条件一般，位置适中
- 仙峰寺：性价比较高
- 雷洞坪：条件较好，靠近金顶

### 山顶住宿
- 金顶大酒店：条件最好
- 金顶招待所：经济实惠
- 需提前预订
